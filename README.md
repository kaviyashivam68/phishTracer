#Phishing Detection of Email

A simple web-based tool to detect phishing emails using **HTML, CSS, and JavaScript**. It analyzes email content for suspicious links, urgent language, and common phishing patterns.

#Features
- Detects suspicious links  
- Highlights urgent or threatening language  
- Checks for untrusted sender patterns  
- Neon-themed dark interface  
- Runs completely in the browser  

#How to Use

1. Open `index.html` in your browser.  
2. Paste the email content in the input box.  
3. Click **Check Email** to see if it is suspicious.
